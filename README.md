# Staircase-Light

Staircase light with two PIR Sensor, one on top of the Staircase and other on the bottom. <br>
If you go from the bottom up, the light goes from the bottom up.
And from the top exactly the other way.

# Components 
- PIR sensors
- Arduino 
- Brightness sensor
- WS2812b LED strips
- LED Profil
- Power supply
